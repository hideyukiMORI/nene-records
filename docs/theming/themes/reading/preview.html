狭さがつくる、親密な読書体験

編集部 Typography Notes

本を読むという行為は、本来とても親密なものです。手のひらに収まる紙面、視線が迷わずに次の行へ落ちていく一定の幅、そして指で送るページの余白——Reading テーマは、画面の上にその親密さを取り戻すことを起点にしています。広く敷き詰めるのではなく、あえて狭く組む。空気のような余白を版面の左右に残すことで、読み手と文章のあいだの距離を縮めます。

版面の幅は 1008px まで絞り、本文の一行は約 66 文字に整えました。行間は 1.74 までゆるめ、本文には serif の Source Serif 4 を据えています。見出しまで同じ serif で揃えることで、ページ全体がひとつの声で語りかけるような、出版物の落ち着きが生まれます。

読みやすさは、足し算ではなく引き算でつくられる。装飾を削り、余白を残し、文字だけを残す。

罫線は、髪の毛のように繊細に

区切りは、囲うのではなく、一本の細い罫で示します。カードもサイドバーも枠で閉じず、余白とヘアラインだけで領域を分ける。過度な装飾やモノスペースの表記は使わず、ラベルには控えめな小さなキャップスだけを添えます。静けさそのものが、このテーマの意匠です。

  • 本文・見出し — Source Serif 4（セルフホスト同梱）
  • ラベル — Saira Semi Condensed の小さなキャップス
  • コード — JetBrains Mono

ダークでも、紙の温度を保つ

暗いモードでは、冷たい黒ではなくセピア寄りの近黒を地にしています。アクセントは同じブラウンの色相のまま明度だけを持ち上げ、リンクは この通り 静かに効きます。--measure や --leading-body、そして版面の幅は、モードに依らず一定に保たれます。
